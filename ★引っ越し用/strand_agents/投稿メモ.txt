タイトル
最近流行りの「Strands Agents」でAIエージェントを作成してみました！

内容
Strands Agentsとは、
AWSから公開されたOSSのAIエージェントSDK（25年5月リリース）
LangChainなどの既存フレームワークと比べて少ないコード（python）でエージェントを開発できる
従来のアプローチのように事前にワークフローを定義する必要がなく、モデルとツールとプロンプトだけ指定すればOK
マルチエージェント（Agent as Tools、swamなど）も実装可、最近流行りのMCPにも対応
詳細はこちら
Strands Agents – オープンソース AI エージェント SDK の紹介 | Amazon Web Services ブログ
今回は下の記事に基づいてAWSコスト最適化エージェントを作成しました。https://tech.layerx.co.jp/entry/2025/10/03/180000
記事と違う点はAWS Knowledge MCPサーバーを追加しているところだけです。
実際の構成は以下になります。
ツール
use_aws 
AWS の各種 API をコールするツールです。AWSの料金情報の取得や各サービスの利用状況取得に用います。
tavily_search
Web 検索ツールです。コスト削減の提案に用いるアイデアを検索する用途で使用します。
APIキーが必要になります。メールアドレス登録すれば、月1000回まで無料でコールできます。
current_time
現在日時取得ツールです。正確に現在日時を把握するために利用します。
AWS Knowledge MCPサーバー　
最新のドキュメント、コードサンプル、その他の公式AWSコンテンツを提供するリモートMCPサーバー
モデル
claude-3-5-sonnet
プロンプト
省略。app.pyをご確認ください。
ちなみに、今回はCLIベースですが、Bedrock AgentCoreというサービスを活用することでデプロイする（AWS上のサーバレス環境にエージェントをデプロイする）ことも可能です。
